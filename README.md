# ensf-338-term-project

### First time setup

#### Windows:
```
setup_env.bat
python tui_app.py
```

#### macOS/Linux:
```
bash setup_env.sh
python tui_app.py
```

### Running after setup

#### Windows:
```
.venv\Scripts\activate
python tui_app.py
```

#### macOS/Linux:
```
source .venv/bin/activate
python tui_app.py
```
